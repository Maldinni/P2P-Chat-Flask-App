English 
P2P Chat Flask App is a simple real-time messaging app built with Flask. 
It lets users sign up, log in, and chat directly with each other in a peer-to-peer 
setup. With Flask and Socket.IO, this app creates a cool, interactive space for 
messaging. It’s a great way to see how Flask can be used for a modern, real-time 
chat app. Perfect for anyone looking to build a simple and effective chat system!
